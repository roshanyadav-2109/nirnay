# Sample data — India-styled

Generated by `npm run generate-mocks` (`scripts/generate-mock-data.mjs`).

Two complete tenders + 10 bidders are bundled, modelled on real Indian
government / banking / certification document conventions.

│   ├── bidder-03-national-infrastructure-corp/(8 PDFs)

## Tender A — CRPF Border Outpost Construction

Estimated cost Rs. 8 Crore, EMD Rs. 16 Lakh.

| Bidder                              | Verdict       | Why                                                              |
|------------------------------------|---------------|------------------------------------------------------------------|
| Sharma Construction Pvt Ltd        | eligible      | All thresholds met cleanly; bonus ISO 14001 + paramilitary work |
| Gupta Builders                     | not_eligible  | Turnover Rs. 3.4 Cr (< Rs. 5 Cr), solvency Rs. 1.5 Cr           |
| National Infrastructure Corp       | needs_review  | ISO 9001 expires on bid date; FY 2024-25 audit pending          |
| Apex Constructions Ltd             | eligible      | Strong financials; 7 paramilitary / defence projects             |
| Metro Build Solutions              | not_eligible  | ISO 9001 expired 2025-06-30 (before bid date 2026-04-15)         |

**Document set per bidder:** cover letter, CA turnover cert, solvency cert,
GST cert, ISO 9001 (and 14001 for Bidder 01), PAN card, project list, EMD bank
guarantee.

## Tender B — AIIMS New Delhi · 50 Adult Mechanical Ventilators

Estimated cost Rs. 12 Crore, EMD Rs. 24 Lakh, 5-year CMC.

11 eligibility criteria covering CDSCO licence, ISO 13485, IEC 60601 testing,
turnover ≥ Rs. 10 Cr, net worth ≥ Rs. 5 Cr, 3+ similar supplies, 5+ years
experience, GST, 10+ city service network, ISO 9001 (preferred), Make-in-India
(preferred).

| Bidder                              | Verdict       | Why                                                                          |
|------------------------------------|---------------|------------------------------------------------------------------------------|
| MediTech Systems Pvt Ltd            | eligible      | Clean OEM with 12 yr history; CDSCO, ISO 13485, IEC 60601, 22-city network |
| Lifeline Healthcare Solutions       | not_eligible  | No CDSCO MD-9 (importer only), no IEC test report, turnover Rs. 7.2 Cr      |
| Pinnacle Medical Devices            | needs_review  | CDSCO licence expired 4 days before bid date; IEC test report expired 12 days before |
| Jeevan Bio-Medical Pvt Ltd          | eligible      | 15-year OEM, Rs. 22 Cr turnover, 28-city network, DPIIT-registered local supplier |
| Universal Hospital Equipment Co.    | not_eligible  | Turnover Rs. 6.8 Cr (< Rs. 10 Cr), 7-city network (< 10), no Rs. 50L+ supply |

**New document types** in healthcare bidders (beyond construction):

* **CDSCO Manufacturing Licence** — Form MD-9 with licence number, expiry, authorised devices, manufacturing site, DG (India) signature, CDSCO seal
* **ISO 13485:2016** — Medical-devices QMS certificate (rendered via the same ISO template)
* **IEC 60601-1 / 60601-2-12 Compliance** — NABL-accredited test report
* **PAN-India Service Network Statement** — table of cities with biomedical engineers and SLAs
* **ISO 9001 (preferred)** for some bidders

## Demo mode pre-cached verdicts

`src/lib/demo-cache.ts` ships pre-computed verdicts for all 10 bidders, so the
dashboard runs evaluations in 1–2 seconds when **Demo mode** is on (default for
new visitors). Disable Demo mode in Settings to call Gemini live.

Bidder names are matched fuzzily — typing the name approximately ("Sharma
Construction" / "MediTech Systems Pvt Ltd" / "Pinnacle Medical Devices") will
hit the right cache entry.

To re-generate from scratch: `npm run generate-mocks` (drops & rewrites all 90+
PDFs in this folder).
