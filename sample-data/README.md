# Sample data — India-styled

Generated by `npm run generate-mocks` (`scripts/generate-mock-data.mjs`).

These PDFs simulate a CRPF civil construction tender + 5 bidders, modelled on
real Indian government / banking / certification document conventions:

* CRPF Notice Inviting Tender (NIT) with file no., bilingual headings, Ashok-style seal, signature block, `eprocure.gov.in` reference
* Bidder cover letter on company letterhead with reference number, subject line, signature + company seal
* CA-issued turnover certificate (UDIN, FRN, ICAI seal) with FY-wise turnover table
* Bank-issued solvency certificate with bank-coloured letterhead, big amount box, branch manager seal
* GST REG-06 registration certificate with tricolour band, `services.gst.gov.in` reference
* ISO 9001:2015 certificate with certifying body, scope, three-date block (issue/expiry/original), accredited body seal
* Income Tax PAN card mimicking the actual photo + signature layout
* List of similar projects in tabular format with Sl. No., name, value, client, completed-on
* Bank guarantee for EMD with BG number, beneficiary, validity, branch seal
