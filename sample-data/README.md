# Sample data

These PDFs are auto-generated by `npm run generate-mocks` (script: `scripts/generate-mock-data.mjs`).
They simulate a CRPF construction tender + 5 bidders, tuned so each bidder produces a known verdict.

## The mock tender

`mock-tender.pdf` — "Construction of Border Outpost Building for CRPF", Rs. 8 Cr.

Eligibility criteria embedded:

| Code | Category   | Rule                                       | Mandatory? |
|------|-----------|--------------------------------------------|-----------|
| C-1  | Financial | Annual turnover ≥ Rs. 5 Cr in last 3 FYs   | Yes       |
| C-2  | Financial | Solvency certificate ≥ Rs. 2 Cr            | Yes       |
| C-3  | Financial | EMD of Rs. 16 Lakh submitted               | Yes       |
| C-4  | Technical | ≥ 3 similar projects ≥ Rs. 2 Cr in 7 yrs   | Yes       |
| C-5  | Technical | ≥ 5 years of construction experience       | Yes       |
| C-6  | Compliance| Active GST registration                    | Yes       |
| C-7  | Compliance| ISO 9001:2015 valid as on bid date         | Yes       |
| C-8  | Document  | PAN Card                                   | Yes       |
| C-9  | Compliance| ISO 14001:2015                             | Preferred |
| C-10 | Technical | Paramilitary/defense construction history  | Preferred |
